# SNHU-IT-145-Grazioso-Salvare

> Coursework for IT-145 at Southern New Hampshire University (SNHU)

A Java rescue animal management system built with OOP principles. Uses a `RescueAnimal` base class with `Dog` and `Monkey` subclasses to handle intake, reservation, and reporting. Features validated input, training status tracking, species eligibility checks, and color-coded console feedback via a menu-driven interface.
